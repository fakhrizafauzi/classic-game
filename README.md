# 🎮 Dot Matrix Arcade

A premium, retro-style multi-game arcade system built with **React**, **TypeScript**, and **Framer Motion**. Inspired by classic handheld consoles, it features a unified input system, dynamic themes, and 6 highly-polished classic games.

## 🕹️ Included Games

1.  **Tetris**: Complete logic with rotation, hard-drop (DOWN), and next-piece preview.
2.  **2048**: Smooth merging with 10 high-contrast tile levels and high-score tracking.
3.  **Minesweeper**: Optimized 8x8 grid for mobile, featuring vertical centering and cell-shading.
4.  **Nonogram (Picross)**: Logic puzzle with auto-solving hints (gray out clues).
5.  **Sudoku**: Full 9x9 logic grid with quick A/B number increment/decrement.
6.  **Tic Tac Toe**: Play against an AI opponent with randomized starts and win-line highlighting.

---

## ✨ Key Features

-   **Responsive Layout (60% Screen)**: Specially optimized for mobile. The game bezel occupies 60% of the phone's height for maximum visibility.
-   **Color Themes (10 Options)**: Toggle through 10 iconic handheld colors (Classic Gray, Atomic Purple, Cyber Pink, etc.) using the **SELECT** button.
-   **Universal Input**: Seamlessly switch between keyboard, touch-screen gamepad, and mouse.
-   **Smooth UI/UX**: Native `scrollIntoView` for menu navigation and `framer-motion` for in-game transitions.
-   **Dot Matrix Aesthetic**: Authentic 4-color palette feel with pixel-grid overlays.

## ⌨️ Controls

### Keyboard
| Key | Action |
| :--- | :--- |
| **Arrow Keys / WASD** | D-Pad Navigation |
| **X / K / Space** | **A Button** (Confirm / Rotate / Fill) |
| **Z / J** | **B Button** (Back / Flag / Cross) |
| **Shift** | **SELECT** (Change Theme / Back to Menu) |
| **Enter** | **START** (Play Game / Reset Game) |

### Virtual Gamepad (Screen)
-   **D-Pad**: Direct movement in each direction.
-   **A Button**: Main action / confirmation.
-   **B Button**: Secondary action / Cancel.
-   **SELECT**: Cycles through themes in Menu, or goes back to Menu during gameplay.
-   **START**: Starts or resets the current game.

## 🚀 Getting Started

    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```
